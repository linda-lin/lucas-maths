# Lucas's Apps

Two single-file HTML apps for Lucas — no build step, no dependencies, just open in a browser.

## LH Math Practice (`index.html`)

Quick-fire maths quiz with auto timer and smart scoring.

- **Operations**: addition, subtraction, multiplication, division (with optional remainders)
- **4 difficulty levels**: L1.1, L1.2, L2, L3
- **Correction round**: retry wrong answers until all correct
- **History**: last 50 sessions saved in localStorage

## Lucas's Todo Board (`lh-todos.html`)

ADHD-friendly daily task tracker with a reward and progression system.

- **Daily tasks**: parent-configured cards Lucas taps to complete
- **Bonus tasks**: optional extras Lucas can pick to earn more stars
- **XP & levels**: earn XP by completing tasks, level up with celebrations (Star Starter → Rising Star → Super Helper → Task Master → Legend)
- **Reward shop**: spend stars on rewards (gaming time, treats, movie night, etc.)
- **Parent setup**: add/remove tasks and rewards, adjust star balance, configure levels
- **Daily auto-reset**: tasks reset each day, stars and levels persist

## Usage

Open the files directly in a browser, or visit via GitHub Pages:

- [Todo Board](https://linda-lin.github.io/lucas-maths/lh-todos.html)
